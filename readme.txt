COUNTER-STRIKE 2 TOURNAMENT ORGANIZING GUIDE
============================================
This repository is a guide to organizing your own Counter-Strike 2 tournaments.
Contents will be expanded on as time goes on and I learn more about this stuff.

DIRECTORIES
===========
console/ - CS2 console commands to make it all possible
configs/ - .cfg files for quick setup of matches (e.g. warmup, knife round, game settings) [TOURNAMENT STANDARD SETTINGS]
mapveto/ - map pick-ban rules (bo1, bo3, bo5)
opt/ - optional stuff
stream/ - useful information if you want to stream your tournament to the world
